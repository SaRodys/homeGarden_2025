Per installare il progetto homeGarden_2025:
- aprire il terminale e scrivere cd homeGarden_2025-main (oppure cd *percorso cartella\homeGarden_2025-main) 
- scrivere npm install
- scrivere npm install bootstrap axios
- scrivere npm run serve
- copiare l'indirizzo http://localhost:8080/ e incollarlo nella barra di ricerca del browser
